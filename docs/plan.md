# Поточний план

1. Розширити ACF-повторювач `course_projects` полями карток проєктів учнів.
2. Вивести спільну карусель після секції процесу на сторінках курсів.
3. Перевірити ACF, HTML і маршрут `/courses/programming-start/` у DDEV.
4. Заповнити редаговану програму курсу в акордеоні після проєктів учнів.
