# Стрілка між блоками навичок на сторінках англійських курсів

## Мета

Додати на desktop серію з чотирьох маленьких сірих пунктирних стрілок, які візуально проводять користувача від кожної картки навичок до наступної в секції `.english-course-skills`.

## Рішення

- Використати наданий PNG-asset, обрізаний до компактного фрагмента стрілки та збережений як RGBA.
- Підключити чотири копії asset через `li:not(:last-child)::after` в існуючому `assets/css/english-course.css`.
- Розмістити по одній стрілці між кожною парою сусідніх карток.
- Додати `pointer-events: none`, щоб декоративні елементи не блокували взаємодію.
- На вузьких екранах приховати стрілки: картки переходять у вертикальну структуру.
- Не змінювати PHP, ACF, JavaScript або залежності.

## Критерії приймання

1. На desktop чотири маленькі стрілки мають бути сірими, помітними та стояти між сусідніми картками.
2. Картки та їхній текст залишаються поверх стрілки й повністю клікабельні.
3. На mobile стрілка не створює обрізання, горизонтальний скрол або перекриття контенту.
4. Сторінки англійських курсів продовжують використовувати існуючий CSS-файл без нової логіки рендерингу.
