# Розмір карток секції «Чому тисячі батьків обирають Logika»

## Мета

Збільшити картки спільної секції на WordPress-сторінках до розміру з дизайн-референсу, не змінюючи тексти, зображення, порядок карток або мобільну структуру.

## Рішення

Змінити розмір тільки через спільні стилі `.media-section__cards`, `.media-section__card` і `.media-section__card-img`. Перед правкою перевірити фактичні computed-розміри на живому WordPress-маршруті; після правки перевірити головну та сторінку «Про Logika» на desktop і mobile.

CSS-джерелом залишається `source/scss/blocks/sections/media-section.scss`; згенерований CSS у темі має відповідати йому. PHP-шаблони та ACF не змінюються.

## Межі

- Змінюється лише візуальний розмір карток.
- Зберігаються існуючі кольори, нахил, картинки, контент і адаптивні breakpoint-и.
- Не додаються JavaScript, нові залежності або page-specific overrides.
- Несуміжні локальні зміни в worktree не зачіпаються.

## Перевірка

Після зміни потрібно пройти PHP syntax check для дотичних шаблонів, існуючий тест static pages, `git diff --check` і live browser smoke з перевіркою кількості карток та їхніх computed-розмірів.
